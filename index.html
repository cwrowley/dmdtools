Welcome

This project provides a collection of tools useful for computing several variants of Dynamic Mode Decomposition. The package includes a set of tools in python and Matlab, though the python version is more complete at the moment. The toolset can be downloaded as .zip or tar.gz by clicking on the associated icons above.
